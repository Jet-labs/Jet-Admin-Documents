# Graphs
-------------
Saved queries are be used as data source for support graphs by defining label, values, x-axis, y-axis etc. 

![Query](/img/graphs.png)

Below are the supported graph types
- [x] Line graph
- [x] Bar graph
- [x] Pie graph
- [x] Radial graph
- [x] Doughnut graph
- [x] Polar graph
